# MongeulBell Skills for Claude Code

MongeulBell(몽글벨) Unity 게임 개발을 위한 Claude Code 스킬 모음입니다.

## 설치

```bash
# 전체 설치 (3개 스킬 모두)
npx skills add babo1408love-cmd/monglebel-skills

# 개별 설치
npx skills add babo1408love-cmd/monglebel-skills@monglebel-gamedev
npx skills add babo1408love-cmd/monglebel-skills@monglebel-art-style
npx skills add babo1408love-cmd/monglebel-skills@monglebel-3d-to-25d
```

## 스킬 목록

| 스킬 | 명령어 | 설명 |
|------|--------|------|
| **monglebel-gamedev** | `/monglebel-gamedev` | 프로젝트 골든룰 + SOLID 엔지니어링 + VS 범위 |
| **monglebel-art-style** | `/monglebel-art-style` | 두근두근 타워 아트 스타일 가이드 |
| **monglebel-3d-to-25d** | `/monglebel-3d-to-25d` | 3D → 2.5D 전환 파이프라인 |

## 몽글벨이란?

**세로형 모바일 2D 하이브리드 전략 게임**
- 보드 이동 + 타워디펜스 + 카드/주사위 전략
- 귀엽고 따뜻한 정화 판타지 세계관
- Unity C# / 3D 셀셰이딩 → 2.5D 프레젠테이션
